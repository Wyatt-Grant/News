🗞️ News Aggregator

Multi-source RSS feed scraper with AI deduplication

Welcome!

This is a news aggregator that combines articles from 11 different news sources, removes duplicates using AI, and presents them in an easy-to-read format.

📡 Multiple Sources

Articles from Star Phoenix, CBC, NYT, BBC, ESPN, and more

🤖 AI Powered

Smart deduplication and summarization

✨ Beautiful Design

Interactive dashboard with source badges

📊 Multiple Formats

HTML dashboard and CSV export

🚀 Getting Started

  1. 1. Clone the repository from GitHub
  2. 2. Install dependencies: pip install -r requirements.txt
  3. 3. Run the scraper: python scraper.py -m 5 -o docs
  4. 4. View results: Open docs/articles.html
